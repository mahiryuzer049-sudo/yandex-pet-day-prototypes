## Как открыть локально
1. Перейти в `C:\Users\frolo\Documents\Playground\yandex-pet-day-test\site`
2. Запустить:
   `python -m http.server 4173`
3. Открыть:
   `http://localhost:4173`

## Быстрый запуск на Windows-host
- Из папки `C:\Users\frolo\Documents\Playground\yandex-pet-day-test` можно запустить:
